# ⚡ InvoiceFlow - מערכת ניהול חשבוניות ו-CRM

מערכת מודרנית ויוקרתית לניהול מסמכים עסקיים, המבוססת על טכנולוגיות Web מתקדמות. האפליקציה עוצבה במיוחד עבור פרילנסרים ועסקים קטנים שרוצים ממשק מהיר, נקי וחכם.

## ✨ תכונות עיקריות

- **ניהול מסמכים מלא:** יצירת חשבוניות מס, הצעות מחיר, קבלות ותעודות משלוח בממשק אחד.
- **ממשק יוקרתי (UI/UX):** עיצוב מבוסס Glassmorphism עם תמיכה מלאה ב-Dark Mode ו-Light Mode.
- **תיק לקוח חכם:** ציר זמן (Timeline) המציג את כל היסטוריית הפעולות מול כל לקוח.
- **תצוגה מקדימה ריאליסטית:** צפייה במסמך המוגמר לפני שליחה, כולל התאמה מלאה להדפסה ו-PDF.
- **הגדרות עסק:** ניהול פרטי העסק והלוגו שמופיעים על גבי המסמכים.
- **Single Page Application (SPA):** ניווט מהיר וחלק ללא טעינת דפים מחדש.

## 🛠 טכנולוגיות
- **HTML5 & CSS3**
- **Tailwind CSS** - לעיצוב מודרני וגמיש.
- **Font Awesome** - לאייקונים ויזואליים.
- **JavaScript** - לניהול הניווט, חישובים ומצב כהה/בהיר.

## 🚀 איך מפעילים?
פשוט מורידים את הקובץ `index.html` ופותחים אותו בכל דפדפן, או שמשתמשים ב-GitHub Pages כדי להפעיל אותו ככתובת אינטרנט חיה.
